## Como adicionar na Vercel:

1. Acesse: https://vercel.com/new
2. Importe o repositório: barber-manager-pro  
3. Na seção "Environment Variables", adicione as 3 variáveis acima
4. **IMPORTANTE**: Marque as 3 opções para cada variável:
   - ✅ Production
   - ✅ Preview
   - ✅ Development
5. Clique em "Deploy"

## Após o deploy:

O banco SQLite será criado automaticamente. Para criar o usuário admin:

1. Acesse a Vercel Console para seu projeto
2. Vá em Settings > Environment Variables
3. Use a função serverless para criar admin

OU execute localmente:
```bash
npm run create-admin
```
